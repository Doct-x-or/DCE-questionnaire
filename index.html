定义： 所有患者获得适合自己体型和血糖水平的固定食谱，不根据个人情况动态调整。
每餐结构： 1/2蔬菜 + 1/4蛋白 + 1/4低GI主食。
主食建议： 优先杂粮、全谷物、薯类，占主食的一半，减少精制米面。
饮品建议： 白水、无糖茶，不建议含糖饮料和果汁。
加餐建议： 水果1份 / 无糖酸奶1份 / 坚果1小把（三选一）。
早餐示例： 燕麦 + 无糖酸奶 + 鸡蛋。
午餐示例： 杂粮饭 + 鱼/鸡胸肉 + 绿叶菜。
晚餐示例： 红薯 + 豆腐/瘦肉 + 菌菇/西兰花。
常见替换： 白米饭→杂粮饭；白面条→荞麦面；奶茶→无糖茶/白水。
